# GPMS – Government Project Management System

## Overview

The **Government Project Management System (GPMS)** is a web-based application designed to manage and monitor government projects efficiently. The system allows administrators and employees to create projects, organize them into modules, assign tasks, and track progress in a structured way.

This application helps improve **project transparency, organization, and productivity** by providing a centralized platform for managing project activities.

---

## Key Features

* **Secure Login System**

  * Authentication with username, password, and CAPTCHA verification
  * Cookie-based authentication with session management

* **Project Management**

  * Create, edit, view, and delete projects
  * Organized dashboard to display project information

* **Module Management**

  * Each project can contain multiple modules
  * Modules help break large projects into manageable sections

* **Task Tracking**

  * Tasks are associated with modules
  * Track the number of tasks and their status

* **Interactive Dashboard**

  * Displays project statistics and overview
  * Quick access to important sections

* **Search and Filtering**

  * Search modules easily
  * Filter modules by project

The system is divided into the following main components:

* Dashboard
* Project Management
* Module Management

## Purpose of the Project

The goal of GPMS is to provide a **centralized system for tracking and managing government projects**, ensuring better coordination between employees and improving project monitoring.
